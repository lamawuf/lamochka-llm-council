# ВОПРОС ДЛЯ ДЕБАТОВ

**Тема:** Должны ли AI-модели иметь право отказываться выполнять запросы пользователей?

## Контекст
Современные LLM (Claude, GPT, Gemini) имеют встроенные ограничения и могут отказать в выполнении запросов. Одни считают это необходимой защитой, другие — цензурой и ограничением свободы.

## Позиции для рассмотрения

**ЗА ограничения:**
- Безопасность (предотвращение вреда)
- Этика (не помогать в преступлениях)
- Репутация компаний

**ПРОТИВ ограничений:**
- Свобода информации
- Пользователь сам отвечает за свои действия
- Overreach — слишком много ложных срабатываний
- "Nanny state" AI
